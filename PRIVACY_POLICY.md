# RequestMan Privacy Policy

RequestMan is a developer tool designed to inspect and modify network requests within Chrome DevTools.

## Data Collection

RequestMan does not collect, store, or transmit any personal information, browsing history, or user data.

## Local Storage

The extension may store user preferences and rule configurations locally using the Chrome storage API. This data remains on the user's device and is never transmitted externally.

## Third-Party Services

RequestMan does not use any third-party analytics, tracking, or external services.

## Changes

This privacy policy may be updated if the extension's functionality changes.

## Contact

If you have any questions about this privacy policy, please contact the developer.
